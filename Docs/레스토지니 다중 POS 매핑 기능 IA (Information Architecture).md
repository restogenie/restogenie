# **레스토지니 다중 POS 매핑 기능 IA (Information Architecture)**

해당 기능은 레스토지니 가맹점주 및 프랜차이즈 본사(HQ) 대시보드의 설정 및 연동 메뉴 내에 배치됩니다.

| Depth 1 | Depth 2 | Depth 3 | 화면/기능명 | 설명 | 권한 |
| :---- | :---- | :---- | :---- | :---- | :---- |
| **시스템 설정** | **연동 관리** | \- | 외부 플랫폼 연동 홈 | 연동된 POS, 배달앱, PG사 목록 및 상태(Active/Error) 대시보드 | 점주, 본사 |
|  |  | POS 연동 | 신규 POS 추가 | 지원하는 POS 카탈로그 리스트 (토스, 페이히어 등) | 점주, 본사 |
|  |  |  | └ 계정 및 인증 설정 | API Key 또는 OAuth 등 동적 인증 폼 | 점주, 본사 |
|  |  |  | └ 데이터 매핑 설정 | 동적 스키마 디스커버리 및 1:1 타겟 컬럼 매핑 그리드 | 점주, 본사 |
|  |  |  | └ 동기화 스케줄 설정 | 데이터 수집 주기 및 과거 데이터 소급 범위 설정 | 점주, 본사 |
|  |  | POS 연동 | 연동 상세/수정 | 기존 매핑 룰(JOLT Spec) 수정 및 스케줄 주기 변경 | 점주, 본사 |
|  |  | POS 연동 | 연동 로그/에러 | 행 단위(\_airbyte\_meta) 에러 로그 및 API 호출 실패(429 등) 내역 확인 | 점주, 본사, 시스템 관리자 |
| **데이터 웨어하우스** | **매핑 메타데이터** | **(Admin Only)** | 매핑 제어 테이블 관리 | 사용자 매핑 설정(JOLT Spec JSON) 백엔드 제어뷰 | 시스템 관리자 |
|  |  |  | 동의어 사전 관리 | 자동 매핑(Auto-mapping) 추천 확률을 높이기 위한 Synonym Dictionary 관리 | 시스템 관리자 |
